Development

Arethusa is built on the AngularJS javascript web application framework and provides a back-end independent plugin infrastructure for accessing texts, annotations and linguistic services from a variety of sources. Extensibility is a guiding design goal - Arethusa includes tools for automatic generation of new plugin skeletons (Arethusa::CLI) and detailed development guides are currently in progress, with the hopes that others will be able to reuse and build upon the platform to add support for other annotation types, languages and back-end repositories and workflow engines.
